Adobe AIR Seamless Install Badge

This page (badge_installer.html) is the same as the default badge installer
with the following differences:

- uses a secure connections (https)
- sets the height and width correctly for the badge
- removes scroll bars
- removes padding and margins
- turns off scaling of the badge
- fixes positioning of the badge contents
- uses html5 doctype

You can use the following iframe embed code:

<iframe style="width: 220px; height: 132px;"
src="https://www.example.com/badge_installer.html?date=0301"
width="220" height="132"></iframe>

To prevent caching update the date parameter on the end of the iframe src URL.
