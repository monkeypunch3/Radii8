Radiate
==================

    Radiate is an application for easily creating designs and websites.
    
    For detailed information about Radiate please visit http://www.velara3.com/

    Currently supported platforms include:

        Microsoft Windows
        Mac OS X

    
    
Install Prerequisites
---------------------

    Before using this software you must install the following software.
    
    ==================================================================================
    SOFTWARE                                   
    ==================================================================================
    
    Adobe AIR 3.6 or greater.

    ==================================================================================
    
    1) Download and install Adobe AIR at http://get.adobe.com/air/. 


    
Installation
---------------------

    After installing Adobe AIR double click on the Radiate.air file. This will launch
    the full installer. Follow the onscreen prompts to complete the installation.
